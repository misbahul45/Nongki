Ini ide section landing page **Nongki/Ningki Reactive CRM** yang paling cocok. Urutannya saya bikin biar storytelling-nya kuat: dari problem → solusi → fitur juara → demo → harga.

## 1. Hero Section

Fokus: langsung jual value utama.

**Judul:**

```txt
Ubah chat WhatsApp jadi insight bisnis dan action otomatis.
```

**Subjudul:**

```txt
Ningki membantu coffee shop, kafe, dan resto membalas chat pelanggan, mencatat Customer 360, mendeteksi hot lead, menemukan lost order, dan mengirim ringkasan bisnis langsung ke WhatsApp owner.
```

Kanan: image 3D AI assistant + dashboard WhatsApp CRM.

---

## 2. Problem Section

Fokus: masalah UMKM F&B.

**Judul:**

```txt
Chat pelanggan selesai, tapi datanya hilang.
```

Isi card:

```txt
Owner lupa follow-up customer yang sudah tanya harga.
```

```txt
Chat WhatsApp tidak berubah jadi database pelanggan.
```

```txt
Produk yang sering ditanya tidak pernah dianalisis.
```

```txt
Pelanggan potensial hilang tanpa reminder.
```

Ini cocok karena masalah utama project kamu memang: chat pelanggan tersebar, owner lupa follow-up, tidak tahu customer potensial, dan tidak punya CRM. 

---

## 3. Solution Section

Fokus: jelaskan posisi produk.

**Judul:**

```txt
Bukan chatbot. Ini Reactive CRM.
```

Buat 3 kolom perbandingan:

```txt
Chatbot biasa
Hanya membalas pertanyaan.
```

```txt
CRM biasa
Hanya menyimpan data.
```

```txt
Ningki Reactive CRM
Membalas, memahami pola, lalu menyarankan action.
```

Kalimat punchline:

```txt
Ningki tidak hanya menjawab pelanggan, tapi membantu owner mengambil keputusan dari chat harian.
```

---

## 4. How It Works Section

Fokus: alur simple.

**Judul:**

```txt
Dari chat masuk sampai action bisnis.
```

Flow horizontal:

```txt
Customer Chat
→ AI Reply
→ Customer 360
→ Insight
→ Owner Digest
→ Campaign Approval
```

Deskripsi pendek:

```txt
Setiap pesan pelanggan masuk ke sistem, dipahami AI, disimpan ke CRM, dianalisis menjadi insight, lalu diubah menjadi rekomendasi follow-up atau campaign.
```

Fokus: revenue.

**Judul:**

```txt

```

Isi:

