Ini prompt lengkap untuk generate image **hero section kanan**, style cocok dengan UI kamu: gemuk, 3D, playful, Duolingo-like, tapi tetap SaaS/CRM.

```txt
Create a playful 3D hero illustration for a SaaS landing page called "Ningki Reactive CRM".

Scene:
A cute, friendly AI assistant mascot managing WhatsApp customer chats for a small coffee shop business. The mascot is sitting or floating beside a large rounded dashboard panel. The dashboard shows customer chat bubbles, customer profile cards, sales insight cards, hot lead alerts, and a daily owner digest message.

Main visual elements:
- A large rounded WhatsApp chat interface with green chat bubbles
- A CRM dashboard card showing "Customer 360"
- Small insight cards showing:
  - "12 chats today"
  - "3 new orders"
  - "2 hot leads"
  - "Lost order detected"
  - "Promo suggestion"
- A coffee cup icon, croissant, small cafe elements
- A friendly robot/AI mascot with rounded body, expressive eyes, and small headset
- Floating notification bubbles
- A growth card with an approve button
- A WhatsApp digest card being sent to the owner

Style:
Playful 3D clay illustration, soft rounded shapes, chunky UI, Duolingo-inspired, friendly, colorful, clean SaaS landing page, modern startup aesthetic, smooth lighting, soft shadows, glossy plastic/clay material, high quality, cute but professional.

Color palette:
Use soft blue, cyan, purple, green accent, warm cream background. Match these colors:
- primary blue
- cyan secondary
- purple accent
- WhatsApp green
- soft neutral background

Composition:
Right-side hero illustration, centered object, lots of empty space around it, suitable for placing text on the left side of a website hero section. The image should not contain real text that needs to be readable, only abstract UI text blocks and simple short labels if necessary.

Background:
Transparent or very light soft gradient background, subtle floating circles, no clutter.

Mood:
Friendly, smart, optimistic, helpful, energetic, modern.

Camera:
Isometric 3D view, slightly front-facing, 3/4 angle, clean composition.

Quality:
Ultra clean, high detail, 4K, smooth render, soft shadows, rounded edges, no harsh outlines.

Negative prompt:
No realistic humans, no messy layout, no dark corporate style, no sharp brutalist edges, no tiny unreadable text, no complex charts, no scary robot, no cluttered background, no photorealism, no watermark, no logo imitation, no distorted UI, no broken hands.
```

Kalau pakai generator yang support aspect ratio, pakai:

```txt
Aspect ratio: 1:1 or 4:3
```

Untuk hero kanan web, paling aman:

```txt
Aspect ratio: 4:3, transparent background
```
