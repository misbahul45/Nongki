# Modules Documentation

Daftar module dalam API dan tanggung jawab utamanya.

- Manajemen user (Register, Login, Logout).
- Validasi JWT.

## Businesses
- Manajemen workspace/bisnis.

## Customers & Conversations
- Inbox CRM dan manajemen kontak pelanggan.

## Products, Reservations, Orders
- Fitur transaksi bisnis.

## Human Handoffs
- Alur pengalihan dari AI ke Admin.
