# Ningki API Service

Main backend API untuk Ningki WhatsApp AI CRM.

## Responsibilities

- Auth
- Business workspace
- Onboarding
- CRM
- Conversation inbox
- Messages
- Knowledge metadata
- Products
- Reservations
- Orders
- Reminders
- Human handoff
- Notifications
- Internal API for AI Engine
- WhatsApp worker bridge
- Webhook handling
- Spreadsheet integration future
- Payment future

## Tech Stack

- Node.js
- TypeScript
- Fastify
- Prisma
- PostgreSQL
- Zod
- JWT
- Pino

## Env

API membaca root `.env` dari `../../.env`.
